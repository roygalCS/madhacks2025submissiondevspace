## Step 2: Push the Code

After creating the repo, run these commands:

```bash
cd /Users/roygal/madhacks2025submissiondevspace

# Add the remote (if not already added)
git remote add origin https://github.com/roygalCS/madhacks2025submissiondevspace.git

# Push to GitHub
git push -u origin main
```
